# Tide Touhou Little Maid Compatibility Mod

这是一个为Minecraft的Tide MOD和Touhou Little Maid MOD之间提供兼容性的模组，允许女仆使用Tide MOD的特殊钓鱼机制进行钓鱼。

## 功能

- 新增了"Tide钓鱼"任务，女仆可以使用Tide的钓鱼竿进行钓鱼
- 支持Tide MOD的所有钓鱼竿（如Midas Fishing Rod）
- 包含适当的钓鱼动画和音效
- 女仆会寻找附近的水源进行钓鱼
- 兼容原版钓鱼机制和Tide特有的钓鱼体验

## 使用方法

1. 确保已安装Touhou Little Maid和Tide MOD
2. 将本兼容性MOD放入mods文件夹
3. 启动游戏
4. 分配女仆为"Tide钓鱼"任务
5. 女仆将自动寻找水源并开始使用Tide钓鱼机制钓鱼

## 开发

本MOD使用NeoForge 1.21.1开发，通过Touhou Little Maid的扩展API注册新的任务类型。
